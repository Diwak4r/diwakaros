# DiwakarOS

The macOS-desktop experience. An interactive portfolio.

[ Live Demo: https://os.diwakaryadav.com.np/ ]

// Architecture
— React.
— Next.js.
— Tailwind CSS.

// Concept
Not a webpage. An environment.
Draggable windows. Resizable interfaces. A simulated desktop right in the browser. 

// Features
— Window Management: Stack, minimize, close.
— Contextual Themes: Dark and Light.
— Responsive: Adapts from monitors to mobile displays.

// Build
> git clone https://github.com/Diwak4r/diwakaros.git
> cd diwakaros
> npm install
> npm run dev
