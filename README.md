## 🌟 Overview

**DiwakarOS** is an interactive, web-based operating system simulation that serves as my personal portfolio. Designed to mimic the sleek and intuitive macOS environment, it provides visitors with a unique and engaging way to explore my projects, skills, and background.

Instead of scrolling through a traditional webpage, users can interact with "apps," open windows, change settings, and navigate a fully functional desktop environment right in their browser.

## ✨ Features

- 🖥️ **macOS Interface:** A highly accurate recreation of the macOS desktop environment.
- 🪟 **Window Management:** Draggable, resizable, and stackable application windows.
- 🌓 **Dark/Light Mode:** Seamlessly switch between themes, just like a real OS.
- 📱 **Responsive Design:** Optimized for both desktop and mobile devices.
- 📂 **Interactive Portfolio:** View my projects, resume, and contact info through native-feeling apps.

## 🚀 Quick Start

If you'd like to run this project locally:

```bash
# 1. Clone the repository
git clone https://github.com/Diwak4r/diwakaros.git

# 2. Navigate into the directory
cd diwakaros

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Visit `http://localhost:3000` to view the OS in action!
